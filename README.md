# fuck_drcom

在没有网页界面下登录 szu 教工区 drcom 。不保证其他地方可用性。

## 使用方法

### 方法一（推荐）

复制一份 `config_sample.py` ，命名为 `config.py`, 在里面输入账号密码，之后直接 `python main.py` 即可。

### 方法二

使用 `CLI`。

```bash
Usage: main.py [OPTIONS]

Options:
  -u, --username TEXT
  -p, --password TEXT
  -a, --keep_alive BOOLEAN
  --help                    Show this message and exit.
```
